Point in/out in Convex Rectangular

Ref : https://gpgstudy.com/forum/viewtopic.php?t=12374

Convex Rectangular A(1)A(2)A(3)A(4)
Point P

P is in the Convex rectangular
if
n = A(1)A(2) x A(2)A(3)
(A(1)A(2) x A(1)P) . n > 0
(A(2)A(3) x A(2)P) . n > 0
(A(3)A(4) x A(3)P) . n > 0
(A(4)A(1) x A(4)P) . n > 0
