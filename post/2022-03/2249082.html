R-squared in Nonlinear regression

Ref : https://statisticsbyjim.com/regression/r-squared-invalid-nonlinear-regression/
https://www.ktpm.co.kr/m04/down.php?no=1783&file_name=A-%C8%B8%B1%CD%BA%D0%BC%AE.pdf

SST : Sum of Square, Total
SSR : Sum of Square, Regression
SSE : Sum of Square, Error

measured data set as a vector: y = [y1, y2, y3, ..., yN]
regression data set as a vector : y^ = [y1^, y2^, ..., yN^]
error vector : e = [e1, e2, ..., eN] = [y1-y1^, ..., yN-yN^]

Define inner produce (a,b) = a1.b1 + a2.b2 + ... + aN.bN

SST = (y,y) = norm(y)^2
SSR = (y^,y^) = norm(y^)^2
SSE = (e,e) = norm(e)^2

For linear regression,
SST = SSR + SSE
because y = y^ + e
         norm(y)^2 = norm(y^)^2 + norm(e)^2 + 2 (y^,e)
         (y^,e) = 0 : by the least-square fit of y^ (detailed proof is depend on you)

Definition of R2
R2 = SSR / SST (between 0 and 1)


For polynomial nonlinear regression
SST = SSR + SSE
because (y^,e) = 0 : by the least-square fit of y^ (detailed proof is depend on you)


For other types of nonlinear regression
There is no evidence that (y^,e)=0
Therefore, SST =~ SSR + SSE

Jim Forest says that "R-squared is not valid for nonlinear regression"
